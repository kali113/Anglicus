# Errores comunes (ES) / Common mistakes (EN)

ES: Patrones frecuentes en hispanohablantes, usados para personalización.
EN: Common Spanish-speaker patterns used for personalization.

## Gramática / Grammar patterns
| Error | Example | Correction |
|---|---|---|
| Missing subject | "Is raining" | "It is raining" |
| Double negatives | "I don't have nothing" | "I don't have anything" |
| Adjective order | "A car red" | "A red car" |
| Present continuous overuse | "I am wanting" | "I want" |
| Ser/Estar confusion | Mixed usage | Context-dependent |

## Falsos amigos / False friends
| Español | Looks like | Actually means |
|---|---|---|
| Actual | Actual | Current |
| Realizar | Realize | Accomplish |
| Librería | Library | Bookstore |
| Sensible | Sensible | Sensitive |
| Embarazada | Embarrassed | Pregnant |
| Éxito | Exit | Success |
| Carpeta | Carpet | Folder |
| Constipado | Constipated | Has a cold |

## Pronunciación / Pronunciation
- /v/ vs /b/ confusion
- /θ/ (th) sounds
- /h/ silent in Spanish, pronounced in English
- Word stress patterns differ

## Artículos / Articles
- Omitting articles: "I went to school" vs "I went to the school"
- Gender habits from Spanish don't apply

## Preposiciones / Prepositions
- "Depend of" → "Depend on"
- "Consist in" → "Consist of"
- "Married with" → "Married to"

## Uso / Usage
ES: Se usa durante onboarding y para generar ejercicios.
EN: Used during onboarding and for exercise generation.
