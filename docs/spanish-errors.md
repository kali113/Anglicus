# Common Mistakes for Spanish Speakers

> **AI Reference Document** - These patterns are sent to the AI during user onboarding
> to personalize exercises. User-specific errors are tracked locally in the app.

## Grammar Patterns

| Error | Example | Correction |
|-------|---------|------------|
| Missing subject | "Is raining" | "It is raining" |
| Double negatives | "I don't have nothing" | "I don't have anything" |
| Adjective order | "A car red" | "A red car" |
| Present continuous overuse | "I am wanting" | "I want" |
| Ser/Estar → Be confusion | Mixed usage | Context-dependent |

## False Friends (Falsos Amigos)

| Spanish | Looks like | Actually means |
|---------|------------|----------------|
| Actual | Actual | Current |
| Realizar | Realize | Accomplish |
| Librería | Library | Bookstore |
| Sensible | Sensible | Sensitive |
| Embarazada | Embarrassed | Pregnant |
| Éxito | Exit | Success |
| Carpeta | Carpet | Folder |
| Constipado | Constipated | Has a cold |

## Pronunciation

- /v/ vs /b/ confusion
- /θ/ (th) sounds
- /h/ silent in Spanish, pronounced in English
- Word stress patterns differ

## Articles

- Omitting articles: "I went to school" vs "I went to the school"
- Gender habits from Spanish don't apply

## Prepositions

- "Depend of" → "Depend on"
- "Consist in" → "Consist of"
- "Married with" → "Married to"

## How This Is Used

1. **Onboarding**: AI asks user diagnostic questions
2. **Detection**: AI identifies which patterns apply to this user
3. **Storage**: User's weak areas saved locally
4. **Exercises**: AI generates targeted practice from weak areas
5. **Tracking**: User mistakes update their personal profile
